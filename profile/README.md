# Plain Plugins

**WordPress plugins and blocks that look native.**

We make small, focused WordPress plugins that look and feel like part of WordPress core. No upsells, no dashboard takeovers. Block-first, follows WordPress coding standards.

## Conventions

Every Plain Plugins repository follows a [shared naming convention](../CONVENTIONS.md). Block namespaces, hook prefixes, CPT keys, PHP namespaces, and slugs are coordinated across the whole portfolio so plugins compose cleanly.
